O que é JavaScript?

JavaScript é uma linguagem de programação usada para tornar as páginas web interativas. Com ele, é possível adicionar efeitos, mudanças de conteúdo e até jogos na página sem recarregar.

Motivação para a criação do JavaScript

O JavaScript foi criado para deixar as páginas web mais dinâmicas e interativas, permitindo que o conteúdo da página mude sem recarregar.

Execução no Navegador vs. Servidor

O JavaScript pode rodar no navegador, onde interage com a página que o usuário vê, ou no servidor, onde processa dados e gera conteúdo para a página.

O que é Sandbox?

No navegador, o JavaScript é executado em um ambiente isolado (chamado de "sandbox") para evitar que ele acesse ou altere dados do usuário sem permissão.

Objetos Globais

JavaScript tem objetos globais que podem ser usados em qualquer lugar, como:

  • document: Permite manipular o conteúdo da página.
  • console: Mostra mensagens no console para depuração.
  • alert(): Exibe mensagens para o usuário em uma janela pop-up.
